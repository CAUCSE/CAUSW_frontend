## CAUSW Front-End

중앙대학교 소프트웨어학부 동문만 사용할 수 있는 커뮤니티 서비스로, <br/> 단순 소셜 네트워크 기능 뿐만 아니라 사물함 신청, 동아리 신청 및 학생회 사업/행사 신청 등  <br/> 전반적으로 소프트 학생 사회를 하나로 묶어주며, 학생들의 편의를 증진 시켜주는 서비스입니다.
